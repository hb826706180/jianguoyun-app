# API 错误码文档

## 错误码说明

本 API 使用 HTTP 状态码和自定义错误码来表示不同的错误类型。

## HTTP 状态码

| 状态码 | 说明 |
|--------|------|
| 200 OK | 请求成功 |
| 201 Created | 资源创建成功 |
| 204 No Content | 操作成功，无返回内容 |
| 400 Bad Request | 请求参数错误 |
| 401 Unauthorized | 未授权访问 |
| 403 Forbidden | 禁止访问 |
| 404 Not Found | 资源不存在 |
| 409 Conflict | 资源冲突 |
| 429 Too Many Requests | 请求过于频繁 |
| 500 Internal Server Error | 服务器内部错误 |
| 502 Bad Gateway | 网关错误（坚果云服务异常） |
| 503 Service Unavailable | 服务不可用 |

## 自定义错误码

### 通用错误 (1000-1999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 1001 | 参数错误 | 请求参数缺失或格式错误 | 检查请求参数 |
| 1002 | 认证失败 | 用户名或密码错误 | 检查坚果云账号密码 |
| 1003 | 权限不足 | 没有操作权限 | 检查应用密码权限 |
| 1004 | 资源不存在 | 请求的文件或目录不存在 | 检查路径是否正确 |
| 1005 | 资源已存在 | 文件已存在，无法创建 | 使用其他文件名 |
| 1006 | 文件大小超限 | 上传文件超过大小限制 | 检查 config.ini 中的 upload_max_size 配置 |
| 1007 | 网络错误 | 网络连接失败 | 检查网络连接 |
| 1008 | 服务异常 | 坚果云服务异常 | 稍后重试 |

### 文件操作错误 (2000-2999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 2001 | 文件上传失败 | 上传文件时出错 | 检查网络和文件路径 |
| 2002 | 文件下载失败 | 下载文件时出错 | 检查文件是否存在 |
| 2003 | 文件删除失败 | 删除文件时出错 | 检查文件是否被占用 |
| 2004 | 目录创建失败 | 创建目录时出错 | 检查父目录是否存在 |
| 2005 | 文件列表获取失败 | 列出文件时出错 | 检查路径是否正确 |
| 2006 | 文件移动失败 | 移动/重命名文件失败 | 检查目标路径 |
| 2007 | 文件复制失败 | 复制文件失败 | 检查磁盘空间 |

### 回收站错误 (3000-3999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 3001 | 回收站操作失败 | 移动到回收站失败 | 检查文件是否存在 |
| 3002 | 文件恢复失败 | 从回收站恢复失败 | 检查原路径是否可用 |
| 3003 | 清空回收站失败 | 清空回收站失败 | 稍后重试 |

### 版本历史错误 (4000-4999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 4001 | 版本历史获取失败 | 获取文件历史失败 | WebDAV 协议暂不支持 |
| 4002 | 版本恢复失败 | 恢复历史版本失败 | WebDAV 协议暂不支持 |

### 批量操作错误 (5000-5999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 5001 | 批量上传失败 | 部分文件上传失败 | 检查错误详情 |
| 5002 | 批量删除失败 | 部分文件删除失败 | 检查错误详情 |
| 5003 | 并发数超限 | 并发请求数过多 | 降低 concurrency 参数 |

### 同步备份错误 (6000-6999)

| 错误码 | 错误信息 | 说明 | 解决方案 |
|--------|----------|------|----------|
| 6001 | 备份任务失败 | 执行备份时出错 | 检查源目录权限 |
| 6002 | 同步任务失败 | 执行同步时出错 | 检查网络和路径 |
| 6003 | 任务已存在 | 重复添加任务 | 使用不同任务名称 |

## 错误响应格式

```json
{
  "error": "错误描述信息",
  "code": 1001,
  "details": {
    "field": "具体字段",
    "message": "详细错误信息"
  }
}
```

## 错误处理建议

1. **网络错误**：建议实现重试机制，最多重试 3 次
2. **认证错误**：检查 config.ini 中的账号密码配置
3. **权限错误**：在坚果云官网检查应用密码权限
4. **速率限制**：遇到 429 错误时，等待 60 秒后重试
5. **文件冲突**：检查文件是否被其他程序占用

## 坚果云 API 限制

- 单文件大小限制：免费用户 1GB，付费用户更大
- API 请求频率：建议不低于 1 次/秒
- WebDAV 连接数：建议不超过 10 个并发连接
