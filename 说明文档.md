# 坚果云文件操作 API - 项目说明

## 一、项目简介

基于 Golang 的坚果云文件管理 API 服务，支持文件的增删改查操作。

## 二、项目规划

### 2.1 项目目标
开发一个基于 Golang 的坚果云文件操作 API 服务，通过 WebDAV 协议实现文件管理功能。

### 2.2 技术栈
| 组件 | 版本 | 说明 |
|------|------|------|
| Go | 1.25.1 | 编程语言 |
| Gin | 1.12.0 | Web 框架 |
| ini | 1.67.1 | 配置文件解析 |
| WebDAV | - | 文件传输协议 |

### 2.3 核心功能
1. 文件列表查询
2. 文件上传
3. 文件下载
4. 文件删除
5. 目录创建

## 六、快速开始

### 6.1 创建配置文件

```ini
[server]
port = 8081
mode = debug

[jianguoyun]
webdav_url = https://dav.jianguoyun.com/dav/
username = your-email@example.com
password = your-webdav-password
```

### 6.2 编译程序

```bash
# Windows
编译 wins.bat

# Linux
编译 linux.bat

# 所有平台
编译 all.bat
```

### 6.3 运行程序

```bash
./build/jianguoyun-app.exe
```

## 七、API 接口

### 基础文件操作

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/files/list?path=/` | 列出目录内容 |
| POST | `/api/files/upload` | 上传文件 (form-data: path, file) |
| GET | `/api/files/download?path=` | 下载文件 |
| DELETE | `/api/files/delete?path=` | 删除文件 |
| POST | `/api/files/mkdir` | 创建目录 (form-data: path) |

### 账户管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/account/info` | 获取账户信息 |

### 回收站管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/trash/list` | 列出回收站 |
| POST | `/api/trash/move?path=` | 移动到回收站 |
| POST | `/api/trash/restore?path=` | 恢复文件 |
| DELETE | `/api/trash/empty` | 清空回收站 |

### 高级功能

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/files/history?path=` | 获取文件历史 |
| POST | `/api/files/batch-upload` | 批量上传 |
| POST | `/api/files/batch-delete` | 批量删除 |

| max_backups | 5 | 最大备份数量 |

## 九、使用示例

```bash
# 列出文件
curl http://localhost:8081/api/files/list?path=/

# 上传文件
curl -X POST http://localhost:8081/api/files/upload -F "path=/test.txt" -F "file=@./file.txt"

# 下载文件
curl -o out.txt "http://localhost:8081/api/files/download?path=/test.txt"

# 删除文件
curl -X DELETE "http://localhost:8081/api/files/delete?path=/test.txt"

# 创建目录
curl -X POST http://localhost:8081/api/files/mkdir -F "path=/new-folder"

# 获取账户信息
curl http://localhost:8081/api/account/info

# 批量上传（并发 5 个）
curl -X POST "http://localhost:8081/api/files/batch-upload?concurrency=5" \
  -F "base_path=/backup" \
  -F "files=@file1.txt" \
  -F "files=@file2.txt"
```

## 十、注意事项

1. **坚果云授权码**: 需在坚果云官网生成应用专用密码
2. **配置文件安全**: `config.ini` 已加入 `.gitignore`，请勿提交
3. **速率限制**: 坚果云 API 有请求频率限制
4. **生产部署**: 建议设置 `mode = release`

## 十一、常见问题

**Q: 如何获取坚果云授权码？**
A: 登录坚果云官网 → 账户信息 → 安全选项 → 应用密码

**Q: 编译失败怎么办？**
A: 运行 `go mod tidy` 更新依赖

**Q: 如何修改服务端口？**
A: 修改 `config.ini` 中 `[server]` 的 `port` 配置项
